# windows 환경

⚡️ 목표

windows 환경에서 실습할 경우 필요한 프로그램을 설치합니다.

  • 명령행 도구
    • GitBash
    • Cmder
  • 테스트

# 명령행 도구

windows는 CMD를 기본 명령행 도구로 제공합니다. CMD는 macOS, linux에서 사용하는 쉘과 사용법이 다르기 때문에 실습을 진행하기 위해서 별도의 명령행 도구를 사용합니다.

TIP

CMD로 kubectl을 사용할 수 있지만 실습 편의상 쉘을 통일합니다.

GitBash 또는 Cmder를 설치합니다.

# GitBash

git (opens new window)을 설치하면 같이 설치되는 GitBash를 이용하여 실습을 진행합니다.

# Cmder

Cmder (opens new window)를 다운받아 *nix 스타일 쉘을 이용할 수 있습니다.

# 테스트

ls, curl등의 명령어를 입력해보세요.
